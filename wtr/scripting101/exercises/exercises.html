Scripting for Testers

Timeclock Lab Exercises

Lab 1. Getting Started

Test Timeclock manually and develop test ideas.

Lab 2. Driving Timeclock Using IRB

Use IRB and the WTR toolkit to drive the Timeclock application.

Lab 3. Writing Scripts

Create simple scripts.

Lab 4. Verifying Results

Check the expected results in your test scripts.

Lab 5. Harnessing Tests

Collect your tests in a test harness. Use assertions.

Test the web services interface to the Timeclock application. This lab that is usually not included in the one-day Scripting for Testers class.
