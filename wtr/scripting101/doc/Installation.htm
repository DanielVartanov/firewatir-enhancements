Scripting for Testers
Installation Instructions

These instructions describe how to install the software we’ll use for our Scripting for Testers class.

During the morning, you will be using Ruby (a programming language) and the IE Controller (a library) on your machine to test server software running on your instructor’s machine. In the afternoon, you will be using Ruby to directly test the web services on the server software. For this, you will need the server software (Timeclock) to be running on your machine. Timeclock is written in Ruby.

Hardware Requirements

The class software runs on Windows machines. We recommend Windows 2000 or Windows XP, but believe that it will also work on Windows NT and Windows 98.

The IE Controller software requires Internet Explorer and therefore only runs on Windows. The Timeclock server was developed on a Macintosh, and we believe that it can also run on any Unix/Linux machine.

Installing the Software

The software you need to install is contained both in the zip file available for downloaded before the class and in the install directory of the CD given to students at the beginning of class. You can use either source.

Ruby

The Ruby installer is ruby168-8.exe.

1.      Double-click this exe file to install.

2.      Select the default installation options (choose all options; install in c:/ruby).

3.      If you are installing as Administrator on Windows 2000 or Windows XP, you do not need to restart your computer. Otherwise, you should.

Ruby Libraries

You will need to install two Ruby libraries.

·        drb-1.3.6          (Distributed Ruby)

·        iec-1.13           (Internet Explorer Controller)

The same basic procedure can be used to install each of these libraries (or “packages”).

1.      Start up a command shell. (Start-> Run, enter cmd.)

2.      Change the current directory (cd) to the one containing the utility.

3.      Type ruby install.rb

Timeclock Server

Copy the timeclock folder on to your c:\ directory.

You'll do all the lab work in this folder:

c:\timeclock\tutorial

The script you use to run the program under test assumes that c:\temp exists. If it doesn't, please create it. (Or change the script. Ask us for help if you need it. The scripts will be read-only after copying them from the CD.)
